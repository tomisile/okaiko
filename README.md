## Endpoints

**Login** - Themed authentication with Benin palace imagery background and forgot password flow

**Dashboard (`/dashboard`)** - Overview with 4 key metric cards, user growth chart, sales by category, and recent transactions

**Users (`/users`)** - Searchable/paginated table with ban/activate, edit modal, and CSV export

**Products (`/products`)** - Product listings with category filters, status management (approve/reject), and bulk operations

**Categories (`/categories`)** - Grid management of Edo-native categories with cultural motifs and item counts

**Transactions (`/transactions`)** - Complete transaction management with dispute resolution, revenue tracking, and fee calculation

**Festivals (`/festivals`)** - Create/manage seasonal Edo festivals (Igue, Ugie Oro, Edo Day) with discount settings and notifications

**Analytics (`/analytics`)** - Advanced reporting with pie charts (sales distribution), bar charts (user demographics), and top sellers table

**Settings (`/settings`)** - Tabbed configuration for theme customization, platform fees, payment gateways (Paystack, Flutterwave), and API integrations

**Profile (`/profile`)** - Admin personal settings with password change functionality
